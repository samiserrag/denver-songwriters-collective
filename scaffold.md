- Next.js 15 (App Router + RSC)
- TypeScript
- Tailwind CSS
- shadcn/ui (default components)

- Everything must be clean, modular, and AI-extensible.

Create only the initial folder + config + base pages:
- app/layout.tsx
- app/page.tsx with a placeholder “Open Mic Drop” hero section

Do not create database tables yet.

Only produce:
- Clean project skeleton
